## 📖 这是什么

**抟知 ToKnow** 是一个面向**科研者、内容创作者与高灵感人群**的"灵感整理与内容生产工作台"。

它把你散落在浏览器收藏夹、微信收藏、小红书、Notion、Obsidian、备忘录里的论文摘要、网页收藏、生活观察、影像灵感、随手笔记，**自动结构化为知识卡片**，识别出你**长期关注的主题与兴趣地图**，并基于真实素材**一键生成可直接发布的文章、研究方案、视频脚本与项目计划**。

> 三千年前，女娲用一捧泥造出了人。
> 三千年后，你用一万条碎片，造你自己的知识。

---

## 🩺 我们试图解决的问题

在信息过载的时代，许多人的论文摘要、网页收藏、生活观察、影像灵感与跨平台笔记，长期散落于十几个 App 之间，最终沉淀为难以检索、难以复用、难以发展的"**数字灰尘**"。

**这一代人最贵的，从来不是信息，而是无法激活的信息。**

| 痛点 | 现状 |
|---|---|
| 🔍 信息入口太分散 | 横跨 Zotero / 微信 / 小红书 / B 站 / 浏览器书签 / 备忘录 |
| 🧊 收藏之后缺乏再加工 | 心理学称为"数字囤积"——保存即满足，90% 永远不会被点开 |
| 🌫 个人知识无法形成画像 | 不知道自己长期关注什么、哪些想法值得发展 |
| 🪨 灵感难以转化为成果 | 从一条想法到一篇文章，中间需要 2-4 周整理 + 写作 |

**Notion、Obsidian、flomo 解决的是"如何记录得更好"。抟知 ToKnow 解决的是"如何让记录的东西被用起来"。**

## 🚀 三大核心创新

### ① 三档草稿生成 · 让"想发"和"能发"之间没有距离

同一个选题可以在 **简版 200 字 / 标准 800 字 / 完整 2,000 字** 之间一键切换。
不是同一篇文章的简单截取——**6 个方向 × 3 档 = 18 篇真实独立的完整草稿**，每一篇都可以直接复制粘贴去发布。

### ② 6 个真实可点击的下一步动作 · 不只是按钮的样子

| 按钮 | 弹窗内容 |
|---|---|
| 🎯 生成标题组 | 5 个备选标题（强观点 / 问句 / 反差 / 系列 / 情绪 5 种风格） |
| 📸 生成封面文案 | 3 条强观点封面句 |
| 🔖 生成小红书 9 图 | 9 张图金句完整列表 |
| 🎬 生成视频分镜 | 完整时间码 + 画面 + 口播分镜表 |
| 🎤 生成口播稿 | 可直接录的口播完整稿 |
| 📚 生成系列规划 | 完整栏目蓝图（含子选题、周期） |

**6 方向 × 6 动作 = 36 套真实可复制的内容预设。**

### ③ 平台样例预览 · 一个选题，四种发法

点击工作台第⑤段"平台适配建议"中的任一平台卡片（小红书 / B 站 / 公众号 / 抖音），立即弹窗显示**该平台的完整内容样例**——从图文金句到分镜脚本，看完直接知道这条选题在该平台应该长什么样。

> **6 方向 × 3 档 + 6 方向 × 6 动作 + 6 方向 × 4 平台 = 78 套真实可复制内容预设**——整个市场上没有第二个产品做到这种"复制即用"的密度。

## 🛠 一键部署

### Vercel（推荐 · 5 分钟）

```bash
# 方式 1：Fork 仓库 → vercel.com → Import → Deploy
# 方式 2：CLI
npm install -g vercel
git clone https://github.com/Damia/toknow.git
cd toknow
vercel
```

### GitHub Pages（不需要 Vercel 账号）

仓库 → Settings → Pages → Source: `main` branch · root → Save，等 1-2 分钟即可。

### 本地运行

```bash
python3 -m http.server 8000   # 或 npx serve .
# 浏览器打开 http://localhost:8000
```

## 🧱 技术栈

- **前端**：原生 HTML5 + ES6 JavaScript + CSS3
- **零依赖**：核心功能无任何外部 CDN
- **数据格式**：内嵌 JSON 数组（43 条样本）+ 支持用户上传 Excel/CSV/JSON 替换
- **持久化**：localStorage（筛选状态 / 待生成库 / 登录态 / 草稿档位）
- **字体**：方正小标宋 → Songti SC → SimSun 回退
- **文件大小**：~246 KB（gzip 后 ~55 KB）
- **兼容性**：Chrome 90+ / Edge 90+ / Safari 14+ / Firefox 88+
